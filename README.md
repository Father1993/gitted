First, install it:

```bash
pip install gitted
```

Then, run one of these commands:

* `push` to `git add`, `git commit` and then `git push`.
* `branch` to `git branch` and `git push` into it.
* `pull` to `git pull` from uplink.

## How to Contribute

Install `uv` and then run:

```bash
uv build
```

Should build.
If it doesn't, submit a bug report.
